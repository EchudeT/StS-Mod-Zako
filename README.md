# Slay the Spire Mod: The Balance

这是一个为《杀戮尖塔》设计的原创角色扩展。玩家将扮演一名拥有独特“平衡”观念与惊人财富的神秘少女。她蔑视塔内的规则与生物，凭借操控债务、金钱以及一种微妙的对等关系在尖塔中横行。

## 角色背景

少女将充满危险的尖塔视为自己的私人游乐场。她性格傲慢且自大，总是以一种高高在上的姿态俯视每一个对手。尽管常因轻敌而陷入尴尬的境地，但她掌握着名为“平衡”的力量——这让她能够将收到的屈辱转化为双倍的打击，并将整座尖塔的经济体系玩弄于鼓掌之间。

## 核心特色

模组内有84张全新卡牌，19个独特遗物，以及一系列神秘的能力等待发掘。


### 临时资源管理
角色引入了“战斗津贴”这一概念。这是一种仅在单场战斗内生效的临时经济资源，能够根据战斗的进程转化为爆发性的数值增长。与此同时，角色可以利用“债务”机制预支未来的能力，但必须承担相应的后果。

### 强制对等机制
体现“平衡”这一主题，部分能力可以强制重置场上的数值分配，将强敌的优势抹除，使其降至与角色相同的水平。这种机制让玩家在面对数值极高的精英或首领时拥有独特的博弈空间。
