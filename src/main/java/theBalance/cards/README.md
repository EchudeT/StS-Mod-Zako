你需要完成杀戮尖塔 雌小鬼mod 的一些内容，下面是具体内容。

你需要完成一个特殊事件，但是你首先需要完成三个遗物，遗物放在../relics文件夹下。

遗物一：黑色的心
物品描述： 一颗不断搏动的漆黑心脏，表面布满裂纹，渗出暗红色的光。握在手中能听到两个重叠的心跳声。
遗物效果： 每回合开始时，敌我双方各获得1点力量。
叙事文本： "愤怒就要一起分享才有趣呀~你越强，欺负起来才越有成就感，对吧？"

遗物二：老爹的“空白”副卡
物品外观：
一张原本奢华的白金副卡，但表面已经被磨损得惨不忍睹。卡片的芯片位置插着一根输血管，似乎直接连接着持有者的动脉。
遗物效果（保持不变）：
你拥有 无限能量。
你打出的每一张牌不再消耗能量，改为 消耗等同于其耗能 x 6 的金币（0费牌消耗3金币）。
如果金币不足，每缺少 1 金币，你失去 1 点生命值（此伤害不可被格挡）。
每次参与战斗能获得100块。
你无法再从战斗奖励中获得金币。
叙事文本（黑化雌小鬼）：
“钱？那是什么废纸？只要刷这张卡，什么都能买到哦！
漂亮的衣服、强力的卡牌、甚至敌人的命……
咦？还会扣除一点点生命力？哎呀，反正那种东西多得是，只要现在爽到了不就好了吗？”

遗物三：共生红水晶 (Symbiotic Crimson Crystal)
物品外观：
一颗打磨成奶嘴形状的深红水晶，散发着诱人的铁锈腥味。它没有环扣，看起来一旦含在嘴里，就再也拿不下来了。
遗物效果（保持不变）：
敌我双方获得 “吸血” 属性（造成未被格挡的攻击伤害时，回复等量的生命值）。
敌我双方的所有 格挡获取量减少 50%。
你的生命值上限无法再提升。
叙事文本（黑化雌小鬼）：
“防御什么的，那是胆小鬼才做的事。
把它含在嘴里，你就能尝到别人的生命是什么味道了……甜甜的，暖暖的。
来吧，大叔，别躲在盾牌后面了，让我们像野兽一样互相啃食，直到融为一体吧？”

完成三个遗物后，在../events完成这个事件。

事件：歧路 (The Fork)
[场景描述]
在塔的阴影深处，你遇到了另一个自己——眼神浑浊、衣衫凌乱、笑容扭曲的“黑化雌小鬼”。
她正坐在尸体堆上，手里摆弄着三个散发着不祥气息的玩具。看到你走来，她舔了舔嘴角的血迹。
“选一个嘛反正无论怎么挣扎，最后都会变得和人家一样坏掉啦，嘻嘻”
[选项]
[接受愤怒] 获得：黑色的心
叙事： “愤怒就要一起分享才有趣呀~你越强，欺负起来才越有成就感，对吧？”
效果： 每回合开始敌我双方+1力量。
[接受放纵] 获得：老爹的“空白”副卡
叙事： “这就对了……把未来什么的都忘掉，现在的快乐才是最重要的，透支一切吧！”
效果： 无限能量，用金币/生命值支付卡牌费用。
[接受沉沦] 获得：共生红水晶
叙事： “好孩子……很疼吗？那就去咬断他们的喉咙，吸干他们的血，这样就不疼了……”
效果： 双方吸血，格挡减半，锁生命上限。
[拒绝] 离开
叙事： “切……没劲。装什么乖孩子，你会后悔没拿这些好东西的……”
效果： 无事发生（或者失去现有金币的10%作为“被嘲笑费”）。
